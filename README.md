本项目是基于 [ORB-SLAM3](https://github.com/UZ-SLAMLab/ORB_SLAM3) 改进的视觉惯导定位系统，旨在解决室内无人机在 **低光照** 与 **复杂运动** 等任务场景下的定位退化问题。

本项目是个人本科毕设的算法原型验证，通过从前端感知、后端融合到优化结构三个层面的自适应优化，显著提升了系统在挑战性场景下的精度、鲁棒性与实时性。
由于时间有限，目前工作仅限于支持单目+IMU模式（Monocular-Inertial），并在TUM-VI数据集上进行系统验证。
这些限制不代表方法本身的固有限制——三个自适应模块在算法层面与传感器配置和平台无关。欢迎有兴趣的研究者进行扩展与迁移验证。
将此项目上传至该平台，一为记录自己毕设过程，二为做相关方向研究工作的同志提供一些灵感启发或参考。


## ✨ Key Features

- **三层自适应协同优化**：前端增强、后端融合、滑窗调整，三者构成完整闭环
- **低光照自适应增强**：基于图像质量评估的自适应Gamma校正，按需触发，避免全帧处理的开销
- **观测可信度动态融合**：根据图像质量与机动状态实时调整视觉/惯性权重，提升复杂环境下的鲁棒性
- **运动感知滑窗优化**：根据加速度与角速度信息自适应调整滑动窗口规模，平衡精度与效率


## 📊 Results

在 TUM-VI 数据集具有挑战性的 “Slides” 序列（含极度暗光封闭滑道）上，相比原始ORB-SLAM3：

| 指标 | 原始ORB-SLAM3 | 本方法 | 提升 |
|:---|:---|:---|:---|
| APE RMSE (m) | 0.660 | 0.561 | **↓ 15.09%** |
| APE Mean (m) | 0.639 | 0.542 | **↓ 15.18%** |
| 平均跟踪时间 (ms) | 15.16 | 13.83 | **↓ 8.79%** |
